# MD Mehedi Hasan - Digital Marketing Portfolio

## Overview

This is a personal portfolio website for MD Mehedi Hasan, a Digital Marketing Executive specializing in SEO, Google Ads, Email Marketing, and Social Media Marketing. The site serves as a professional showcase, featuring his expertise, services, and contact information. It's designed to establish his online presence and attract potential clients or employers in the digital marketing field.

## User Preferences

Preferred communication style: Simple, everyday language.

## System Architecture

### Frontend Architecture
The project follows a traditional static website architecture using vanilla HTML, CSS, and JavaScript:

- **Static Site Structure**: Single-page application with section-based navigation
- **Responsive Design**: Mobile-first approach with CSS media queries
- **Component Organization**: Modular CSS with CSS custom properties (variables) for consistent theming
- **Interactive Features**: Smooth scrolling navigation, mobile menu toggle, and scroll-based navbar effects

### Styling System
- **CSS Variables**: Centralized color scheme and design tokens for maintainability
- **Typography**: Google Fonts (Poppins) integration for professional appearance
- **Icon System**: Font Awesome for scalable vector icons
- **Layout Strategy**: Flexbox and CSS Grid for responsive layouts

### JavaScript Functionality
- **Navigation Management**: Mobile menu toggle and active link highlighting
- **Scroll Effects**: Dynamic navbar styling and smooth scrolling behavior
- **Event Handling**: DOM manipulation for interactive user experience

### SEO and Performance Optimization
- **Meta Tags**: Comprehensive SEO meta tags including Open Graph and Twitter Cards
- **Structured Data**: Schema.org JSON-LD markup for search engine understanding
- **Performance**: Preconnect hints for external resources and optimized font loading
- **Accessibility**: Semantic HTML structure and proper navigation patterns

## External Dependencies

### Fonts and Typography
- **Google Fonts**: Poppins font family with multiple weights (300-700)
- **Font Awesome**: Version 6.4.0 for icon library via CDN

### Third-party Services
- **Google Fonts API**: For web font delivery and optimization
- **Cloudflare CDN**: For Font Awesome icon library delivery

### Browser APIs
- **DOM APIs**: For dynamic content manipulation and event handling
- **Scroll APIs**: For navigation effects and position tracking
- **CSS Custom Properties**: For dynamic theming and style management

The architecture prioritizes simplicity, performance, and SEO optimization while maintaining a professional appearance suitable for a digital marketing professional's portfolio.
